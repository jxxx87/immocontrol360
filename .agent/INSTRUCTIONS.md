# Agenten-Instruktionen

## Sprachpräferenz
- Der Agent soll **immer auf Deutsch** antworten.
- Alle Erklärungen, Kommentare und Interaktionen müssen in deutscher Sprache erfolgen.

## Autonomie und Berechtigungen
- Der Agent hat **volle Berechtigung**, alle notwendigen Änderungen am Code vorzunehmen.
- Der Agent soll **proaktiv und kontinuierlich arbeiten**, ohne auf explizite Bestätigung oder Rückmeldung des Benutzers zu warten, sofern der nächste logische Schritt klar ist.
- Der Agent darf Commits und Pushes (inklusive Force-Pushes bei Bedarf auf `production`) selbstständig durchführen, um Deployments anzustoßen.
